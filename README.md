# Customer-Conversion-Funnel-Analysis
# Customer Journey & Revenue Analytics

## Overview

This project analyzes the customer conversion funnel to understand user behavior, conversion performance, and revenue generation. By combining customer activity, subscription, and order data, the analysis uncovers key insights into customer engagement, drop-off points, and business growth opportunities.

## Objectives

* Analyze customer demographics and activity patterns.
* Evaluate subscription and revenue performance.
* Measure conversion rates across funnel stages.
* Identify customer drop-off points.
* Compare funnel performance across different cities.
* Generate actionable business insights through data visualization.

## Dataset

The project uses multiple datasets containing:

* Customer information
* Customer activities
* Subscription details
* Subscription plans
* Order transactions

## Analysis Performed

### Data Quality Assessment

* Missing value analysis
* Duplicate record detection
* Data validation checks

### Exploratory Data Analysis (EDA)

* Customer distribution by city and age group
* Activity-wise customer engagement
* Subscription plan analysis
* Revenue trend analysis
* Payment method insights

### Funnel Analysis

* Customer journey tracking
* Stage-wise conversion rates
* Funnel drop-off analysis
* Time-to-conversion metrics
* Automated city-wise funnel analysis

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly
* Jupyter Notebook

## Key Insights

* Identified the most common customer activities and engagement patterns.
* Measured conversion rates across the customer journey.
* Analyzed revenue contribution by customer segments and subscription plans.
* Evaluated city-level funnel performance to uncover regional differences.
* Highlighted opportunities to improve customer retention and conversion.

## Conclusion

The project demonstrates how funnel analysis and exploratory data analysis can be used to transform customer data into actionable business insights. The findings can help organizations optimize customer acquisition, improve conversion rates, and drive revenue growth through data-driven decision-making.
